# About genroot

genroot is a modern terminal emulator for Android that gives you a full Linux environment right on your device.

## Features

- Full terminal emulation with bash/zsh
- Alpine Linux and Ubuntu support
- Access to a sandboxed Linux environment
- Sudo access without password (in the sandbox)
- Virtual keys for convenient input
- Multiple session support
- Material 3 design with light and dark themes
